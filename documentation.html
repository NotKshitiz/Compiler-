1. Wallet & Contract Functions (11)

connectWallet(privateKey)

Connects to a wallet using the provided private key.

  • privateKey: A secret key to sign transactions and manage wallet access.
connectWallet("0xYOUR_PRIVATE_KEY");

contract(bytecode)

Deploys a contract using the provided bytecode.

  • bytecode: The compiled smart contract code in hexadecimal format.
contract("0xYOUR_BYTECODE");

setOwner(address)

Sets the owner of the contract.

  • address: The Ethereum address to be designated as the owner.
setOwner("0xOwnerAddress");

getOwner()

Returns the currently set contract owner. 

getOwner();

isOwner(address)

Checks if the provided address is the contract owner.

  • address: The Ethereum address to verify ownership.
isOwner("0xAddress");

destroyContract()

Destroys the contract (only executable by the owner). 

destroyContract();

signMessage(message)

Signs a message using the connected wallet.

  • message: The text to be signed for verification purposes.
signMessage("Hello world");

verifySignature(message, signature, address)

Verifies that a message signature was created by the holder of the given address.

  • message: The original message.
  • signature: The cryptographic signature of the message.
  • address: The Ethereum address of the signer.
verifySignature("Hello", "0xSignature", "0xAddress");

getNonce(address)

Gets the transaction count (nonce) for an address.

  • address: The Ethereum address for which the nonce is retrieved.
getNonce("0xAddress");

getContractCode(address)

Returns the bytecode of a deployed contract.

  • address: The contract address whose bytecode is required.
getContractCode("0xContractAddress");

getBlockTimestamp(blockNumber)

Gets the timestamp for a specific block.

  • blockNumber: The number of the block for which to retrieve the timestamp.
getBlockTimestamp(123456);

2. Token Functions (10)

mintTokens(address, amount)

Mints tokens to a specified address.

  • address: The Ethereum address to receive the minted tokens.
  • amount: The number of tokens to mint.
mintTokens("0xAddress", 1000);

transferTokens(from, to, amount)

Transfers tokens from one address to another.

  • from: The sender’s Ethereum address.
  • to: The recipient’s Ethereum address.
  • amount: The number of tokens to transfer.
transferTokens("0xFrom", "0xTo", 500);

burnTokens(address, amount)

Burns a specific amount of tokens from an address.

  • address: The Ethereum address from which tokens will be burned.
  • amount: The number of tokens to burn.
burnTokens("0xAddress", 200);

balanceOf(address)

Retrieves the token balance for a given address.

  • address: The Ethereum address to check for tokens.
balanceOf("0xAddress");

getTokenBalance(tokenAddress, address)

Retrieves the balance of an ERC20 token for a given address.

  • tokenAddress: The ERC20 token's contract address.
  • address: The address for which the token balance is checked.
getTokenBalance("0xToken", "0xAddress");

approve(owner, spender, amount)

Approves a spender to transfer tokens on behalf of the owner.

  • owner: The Ethereum address that owns the tokens.
  • spender: The address that is allowed to spend the tokens.
  • amount: The maximum number of tokens that can be spent.
approve("0xOwner", "0xSpender", 100);

transferFrom(spender, from, to, amount)

Transfers tokens from one address to another on behalf of the token owner.

  • spender: The address executing the transfer.
  • from: The address from which tokens are drawn.
  • to: The recipient’s address.
  • amount: The number of tokens to transfer.
transferFrom("0xSpender", "0xFrom", "0xTo", 50);

getAllowance(owner, spender)

Retrieves the remaining number of tokens that the spender is allowed to transfer on behalf of the owner.

  • owner: The token owner's address.
  • spender: The address allowed to spend the tokens.
getAllowance("0xOwner", "0xSpender");

getTokenDecimals(tokenAddress)

Retrieves the number of decimals used by an ERC20 token.

  • tokenAddress: The ERC20 token's contract address.
getTokenDecimals("0xToken");

getTokenSymbol(tokenAddress)

Retrieves the symbol of an ERC20 token.

  • tokenAddress: The ERC20 token's contract address.
getTokenSymbol("0xToken");

getTokenTotalSupply(tokenAddress)

4. Transaction Functions (6)

sendETH(network, to, amount)

Sends ETH to a specified address on a given network.

  • network: The blockchain network to use (e.g., "linea-sepolia").
  • to: The recipient's Ethereum address.
  • amount: The amount of ETH to send.
sendETH("linea-sepolia", "0xTo", "0.1");

sendToken(tokenAddress, to, amount)

Sends ERC20 tokens to a specified address.

  • tokenAddress: The ERC20 token's contract address.
  • to: The recipient's Ethereum address.
  • amount: The amount of tokens to send.
sendToken("0xToken", "0xTo", "10");

getTransactionStatus(txHash)

Retrieves the status of a specific transaction.

  • txHash: The transaction hash to check.
getTransactionStatus("0xTxHash");

estimateGas(to, amount)

Estimates the gas required for a transaction.

  • to: The recipient’s address.
  • amount: The amount of ETH to be sent (used for estimation).
estimateGas("0xTo", "0.1");

getGasUsed(blockNumber)

Retrieves the total gas used in a specific block.

  • blockNumber: The block number to query.
getGasUsed(123456);

getPendingTransactions()

Retrieves the count of pending transactions in the network. 

getPendingTransactions();

6. Utility Functions (3)

isValidAddress(address)

Validates whether the given string is a proper Ethereum address.

  • address: The Ethereum address to validate.
isValidAddress("0xAddress");

ethToWei(amount)

Converts an amount from ETH to WEI.

  • amount: The amount in ETH.
ethToWei(1);

weiToEth(amount)

Converts an amount from WEI to ETH.

  • amount: The amount in WEI.
weiToEth(1000000000000000000);
